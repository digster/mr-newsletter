# Flet Icons Reference

Flet uses Material Design icons via `ft.Icons.*`. This reference covers commonly used icons.

## Usage

```python
ft.Icon(ft.Icons.FAVORITE, color=ft.Colors.RED, size=24)
ft.IconButton(icon=ft.Icons.ADD, on_click=handler)
ft.ElevatedButton("Save", icon=ft.Icons.SAVE)
```

## Icon Variations

Many icons have variations:
```
ft.Icons.HOME              # Filled
ft.Icons.HOME_OUTLINED     # Outlined
ft.Icons.HOME_ROUNDED      # Rounded corners
ft.Icons.HOME_SHARP        # Sharp corners
```

## Custom Icons

Use icon fonts or SVG:
```python
# Icon font (after loading font)
ft.Text("\ue900", font_family="my-icons")

# SVG as image
ft.Image(src="icon.svg", width=24, height=24)
```
